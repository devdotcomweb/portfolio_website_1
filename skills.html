hard skills or what I have learned so far

I can write simple applications using HTML, CSS, Vanilla JavaScript and React. You can find them on my Github profile.


I have been learning on my own with Udemy. Thanks to this, I was able to develop a learning method, thanks to which I learned the most.
First, I listened to the lessons and took notes. As the instructor went to the project, I looked at the material and understood the code. When the instructor finished it, I printed the code and pasted it into my notebook and took the time to understand the code and make additional notes. I learned from documentation and YouTube videos.
Once I knew what this code came from, I tried to write the code myself, looking at the application's functionalities. On the second or third try, I was able to write the application myself.
Thanks to this learning method, I learned how to search for the necessary information on the Internet and how to write the code myself.
My plan for the near future is to learn next.js. I am probably learning it as you read this.
